# AI Resume Analyzer (Python)

This project analyzes a resume and compares it with a job description to calculate a match score.

## Features
- Upload resume PDF
- Compare with job description
- Show match percentage
- Suggest resume improvement

## Tech Stack
streamlit
pdfplumber
scikit-learn
matplotlib
## Run the project

pip install -r requirements.txt

streamlit run app.py
